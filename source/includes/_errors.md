# Testing Tips
While most Koi nodes rely on bundlers to store consensus data on their behalf, you can also interact directly with the Koi smart contract using the Koi Tools library. In order to do that, you'll first want to visit the <a href="https://faucet.arweave.net/">Arweave Faucet</a> to get some free storage tokens, and then return back here once you have a standard wallet.json RSA file.
